# Conclusiones individuales

El proyecto nos ha permitido comprender el funcionamiento de los procesos en Java y su comunicación de una manera más clara, ya que sabemos la teoria pero no hemos practicado tanto, entender la estructura y leer bien el enunciado han sido procesos clave para resolver lo indicado.

Hemos aprendido entre nosotros a diseñar sistemas concurrentes, evitar hardcode, realizar las constantes de una forma clara y modularizar el código aplicando buenas prácticas.

También entendimos la importancia de documentar correctamente cada fase del desarrollo y de incluir pruebas automatizadas que verifiquen la salida final, tanto en libretas fisicas como realizandolo en documentos markdown.
