# Documentación del proyecto Titanic

Este proyecto simula la gestión de los botes salvavidas del Titanic, un escenario en el que 20 botes debían coordinarse para contabilizar a los supervivientes.  
El objetivo es modelar este proceso en Java usando procesos independientes, comunicación por stdout y generación de informes automáticos en formato Markdown.

Este directorio contiene la documentación dividida en archivos separados.

- [Plan de pruebas](plan_de_pruebas.md)
- [Problemas encontrados](problemas_encontrados.md)
- [Conclusiones individuales](conclusiones.md)
- [Arquitectura (imágenes)](arquitectura.md)
